# Aplicativo Web - TelCol

## Descripción 
Aplicativo web diseñado para permitir a los usuarios agendar citas para la reparación de celulares, consultar el estado de sus equipos y realizar seguimiento al proceso de reparación. El sistema facilita la gestión de clientes, servicios técnicos y control de las reparaciones, mejorando la organización y comunicación entre el taller y los usuarios.

## Objetivo 
Aplicación web para gestionar el registro, diagnóstico y seguimiento de celulares en proceso de reparación.

## Tecnologías utilizadas
- React
- Vite
- JavaScript
- HTML5
- CSS3
- Node.js
- npm
- ESLint
